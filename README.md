# Radar Cultural Inteligente 🎧

Este é um projeto em Streamlit para coleta e análise de tendências culturais com dados simulados do Spotify e YouTube.

## Funcionalidades
- Coleta de dados com exemplo fictício
- Análise de sentimentos (mock)
- Associações com algoritmo Apriori (mock)
- Filtros por fonte, palavra-chave, data e região
- Visualizações com Plotly
- Exportação para Excel, PDF e PNG

## Instalação

1. Clone ou descompacte o projeto:
```bash
git clone https://github.com/seuusuario/radar-cultural-inteligente.git
cd radar-cultural-inteligente
```

2. Instale as dependências:
```bash
pip install -r requirements.txt
sudo apt install wkhtmltopdf
```

3. Execute o Streamlit:
```bash
streamlit run app.py
```

## Publicação no Streamlit Cloud

1. Faça login em [streamlit.io/cloud](https://streamlit.io/cloud)
2. Crie um novo app apontando para o repositório
3. Defina o arquivo principal como `app.py`
